YOPE Website Redesign

Project Overview

YOPE is a Polish natural cosmetics brand that focuses on eco-friendly and low-waste products. I redesigned their website for desktop and mobile, aiming to simplify the interface, reduce clutter, and make the site more accessible for all users.

My Role

Sole UX/UI designer responsible for heuristic analysis, accessibility evaluation, wireframing, and high-fidelity prototyping.

Process

  • Heuristic analysis to identify usability issues
  • Defined project goals and created user stories
  • Benchmarking to gather inspiration from other apps/websites
  • Accessibility analysis: typography hierarchy, button states, WCAG compliance, simplified copy, color contrast, clear error handling
  • Designed information architecture, paper wireframes, and interactive high-fidelity Figma prototypes

Outcome

A clean, accessible, and user-friendly website that aligns with YOPEâ€™s eco-friendly brand identity.

Tools & Methods

Figma, WAVE, Colour Contrast Analyser, Jasnopis, Logios, UX Writing Guidelines, WCAG

GlossoVR

Project Overview

GlossoVR is a virtual reality platform designed for immersive language learning. The project focused on creating an engaging, intuitive, and visually appealing user experience that enhances VR-based education.

My Role

This comparative study presents the results of a user experience investigation conducted during the use of relaxing VR applications. I created game scenario and user stories, and also a simple visualization in Unity. On the basis of the visualization, a prototype was implemented by GlossoVR development team.

Process

  • Desk Research: state-of-the-art review and literature analysis
  • Conceptualisation: research objectives, questions, and hypotheses
  • User Tests & Interviews: tested four VR relaxation games with 12 participants; questionnaires and semi-structured interviews conducted
  • Results Analysis: quantitative analysis of immersion and anxiety; box charts for comparison
  • Conclusions: correlation analysis between immersion and anxiety; summarized study results
  • Visualization & Prototype: game scenario, user stories, Unity mockups, and recommendations
