YOPE Website Redesign

Project Overview

YOPE is a Polish natural cosmetics brand that focuses on eco-friendly and low-waste products. I redesigned their website for desktop and mobile, aiming to simplify the interface, reduce clutter, and make the site more accessible for all users.

My Role

Sole UX/UI designer responsible for heuristic analysis, accessibility evaluation, wireframing, and high-fidelity prototyping.

Process

  • Heuristic analysis to identify usability issues
  • Defined project goals and created user stories
  • Benchmarking to gather inspiration from other apps/websites
  • Accessibility analysis: typography hierarchy, button states, WCAG compliance, simplified copy, color contrast, clear error handling
  • Designed information architecture, paper wireframes, and interactive high-fidelity Figma prototypes

Outcome

A clean, accessible, and user-friendly website that aligns with YOPE’s eco-friendly brand identity.

Tools & Methods

Figma, WAVE, Colour Contrast Analyser, Jasnopis, Logios, UX Writing Guidelines, WCAG

GlossoVR

Project Overview

GlossoVR is a virtual reality platform designed for immersive language learning. The project focused on creating an engaging, intuitive, and visually appealing user experience that enhances VR-based education.

My Role

This comparative study presents the results of a user experience investigation conducted during the use of relaxing VR applications. Four different experiences, distinguished by style and interactivity (Fujii, Real VR Fishing, TRIPP, Nature Treks VR), were tested by 12 participants. I created the game scenario, user stories, and a simple visualization in Unity. Based on this, the prototype was implemented by GlossoVR's development team.

Process

  • Desk Research: analyzed articles and publications on VR for relaxation and anxiety reduction
  • Conceptualisation: defined study objectives, research questions, and hypotheses
  • User Tests & Interviews: analyzed 15 VR apps, selected 4, tested with Oculus Quest 2, conducted questionnaires and interviews
  • Results Analysis: summarized immersion and anxiety metrics, presented averages and comparisons
  • Visualization & Prototype: created game scenario, user stories, Unity visualization, and recommendations
